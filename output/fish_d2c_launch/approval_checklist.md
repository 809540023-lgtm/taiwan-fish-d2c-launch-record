# 人工審核清單

每次上架、發文、群發 LINE、開團、出貨前都要檢查。

## 商品資料

- [ ] 魚種已確認。
- [ ] 產地已確認。
- [ ] 捕撈/養殖來源可說明。
- [ ] 保存型態已確認：冷藏、冷凍、熟食或其他。
- [ ] 每尾重量範圍已確認。
- [ ] 有效日期已確認。
- [ ] 食品業者登錄/加工/分裝資訊已確認。
- [ ] 標示內容已確認。

## 價格與庫存

- [ ] 售價已確認。
- [ ] 運費或免運門檻已確認。
- [ ] 可售庫存已確認。
- [ ] 安全庫存未被挪用。
- [ ] 團購價與零售價不衝突。

## 文案

- [ ] 沒有療效、補身、治病等宣稱。
- [ ] 沒有「最新鮮」「最低價」「零風險」等無法證明用語。
- [ ] 沒有未確認產地或檢驗宣稱。
- [ ] 圖片為真實商品或已標示情境示意。
- [ ] CTA、LINE 連結、下單方式正確。

## 出貨

- [ ] 出貨日已確認。
- [ ] 配送區域可低溫配送。
- [ ] 客人可於到貨時段收貨。
- [ ] 週日、連假、颱風風險已檢查。
- [ ] 包材、蓄冷材、外箱標籤已備妥。

## 對外動作

- [ ] 發文由人工按下發布。
- [ ] LINE 群發由人工按下發送。
- [ ] 私訊回覆由人工確認後送出。
- [ ] 廣告投放由人工設定與確認。
- [ ] 預算調整由人工確認。
